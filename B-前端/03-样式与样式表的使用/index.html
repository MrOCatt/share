    03 样式与样式表的使用

    样式表的引用

    1. 外部引入 
      <link rel="stylesheet" type="text/css" href="mystyle.css" />
    2. 内部引入 
      <style>
  * {
  }
  h1 {
    color: red;
  }
</style>
    3. 行内引入 
      <h1 style="color:blue;">行内引入</h1>

    样式

    语法

    我们如何运用 CSS 来美化我们的样式呢？

    ```css
selector {
    property:style-attribute;
}
```

    • selector 为选择器

    • property 为属性

    • style-attribute 为开发者所想设置的样式属性值

    常用样式属性

    1. 背景

    2. background-color 属性指定元素的背景色。

    3. background-image 属性指定用作元素背景的图像。

    4. background-repeat 属性指定了图片是否在容器里重复显示

      值 作用
      repeat-x 只在水平方向重复
      repeat-y 只在竖直方向重复
      norepeat 不重复

    5. background-attachment 属性指定背景图像是应该滚动还是固定的

      值 作用
      fixed 固定背景图像
      scroll 背景图像应随页面的其余部分一起滚动

    6. background-size 属性

      值 作用
      cover 将图片覆满背景区域

    7. 简写形式 

      div {
  background: #ffffff url("tree.png") no-repeat right top;
}

    8. 边框

    9. border-style 属性指定要显示的边框类型

    10. border-width 属性指定四个边框的宽度

    11. border-color 属性用于设置四个边框的颜色

    12. border-radius 属性用于向元素添加圆角边框

    13. border-方向-样式 用于定义单独边的样式属性

    14. 简写形式 

      div {
  border: 2px solid red;
}

    15. 文本

    16. color 属性用于设置文本颜色

    17. text-align 属性用于设置文本的水平对齐方式

      值 作用
      center 居中
      left 左对齐
      right 右对齐
      justify 每条线都有相等的宽度（适用于报纸）

    18. text-decoration 属性用于设置或删除文本装饰

    19. text-decoration 属性用于设置或删除文本装饰

    20. text-transform 属性用于指定文本中的大写和小写字母

    21. text-indent 属性用于指定文本第一行的缩进

    22. text-shadow 属性为文本添加阴影

    23. 字体

    24. font-family 属性规定文本的字体
    25. font-size 属性设置文本的大小
    26. font-style 属性主要用于指定斜体文本
    27. font-weight 属性指定字体的粗细
    28. 简写 
      p.a {
  font: 20px Arial, sans-serif;
}
